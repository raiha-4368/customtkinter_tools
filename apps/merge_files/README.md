# ファイル結合ツール
## CustomTkinterを使用したファイル結合ツール
選択した2つのファイルを結合するツール
1つ目のファイル末尾に改行がない場合、自動で挿入される
2つ目のファイル末尾に改行がある場合、自動で削除される

## できること
- 2つまでファイルを選択し、そのファイルの記述内容を表示する。2つ選択した場合は改行コード(\n)を挿入し、1つのテキストボックスに表示する。  
- 上記のファイル出力順を入れ替える  
- テキストボックスの編集
- テキストボックスの記述内容をtxt形式でファイルへ出力する。またその際に元ファイルの削除を行う(任意設定可)

## 起動及び使用手順
main.exeファイルの実行
もしくはコマンドプロンプト(プロジェクトルート)で以下コマンドを実行  
python -m apps.merge_files.main  

※python -m はPythonモジュールをスクリプト(実行用ファイル)として実行するためのコマンドラインオプション  

1. "1つ目のファイルを選択"ボタンを押下でファイルを開くダイアログが開きファイルを選択。
テキストボックス欄に選択したファイル内容が表示される。  
2. "2つ目のファイルを選択"ボタンを押下でファイルを開くダイアログが開きファイルを選択。
1.の表示内容に付け足す形でテキストボックスに表示される
3. "ファイルへ出力"ボタンを押下することでファイル保存ダイアログが開き、任意の場所に保存できます。

## txt形式の出力ファイル
デフォルトのファイル名:merge_file_YYYYMMDD_hhmmss.txt

## 簡易テスト
### ■正常系
- 1つ目のファイル選択 → テキストボックスに表示 → ファイル出力
- 2つ目のファイル選択 → テキストボックスに表示 → ファイル出力
- 1,2のファイルを選択 → mergeされた状態でテキストボックスに表示 → ファイル出力
- 1,2のファイルを選択 → 選択ファイル順を逆転ボタン押下 → ファイルパスが入れ替わり、テキストボックスに逆にmergeされた状態のものが表示 → ファイル出力
- 1,2のファイルを選択 → 編集不可ボタンを押下し編集可能にする → テキストボックス欄を編集 → ファイルへ出力

### ■異常系
- 1,2のファイル未選択で選択ファイル順を逆転ボタンを押下 → エラーダイアログ表示  

### ■境界・特殊ケース
- ファイル未選択でファイルへ出力ボタンを押下 → ファイル保存ダイアログが開き保存できる  
- 1つ目のファイルのみを選択し、選択ファイル順を逆転 → path記述欄が切り替わり、先頭に空行が挿入された状態で選択したファイル情報が表示される
- 2つ目のファイルのみを選択し、選択ファイル順を逆転 → path記述欄が切り替わる。テキストボックスには変化なし
