ファイル差分表示ツールです。

## できること
- csvファイルを開き、その内容を表示する(編集不可)  

## 使用技術
- Python
- Custom Tkinter
- Tkinter

## 環境
- Python 3.10 以上(pyファイル)
- Windows(exeファイル)


## 起動及び使用手順
main.exeファイルの実行
もしくはコマンドプロンプト(プロジェクトルート)で以下コマンドを実行  
python -m apps.csv_viewer.main  

※python -m はPythonモジュールをスクリプト(実行用ファイル)として実行するためのコマンドラインオプション  

1. 1つ目のファイルを選択ボタンを押下  
2. 1つ目のファイルを選択ボタンを押下  
3. ツリービューに差分が薄い赤色で表示されます

## 簡易設計
<details>
<summary>簡易設計(折り畳み)  </summary>

main.py  
	∟__init__(初期化、frame生成)  
	∟ create_main_frame(フレーム内の要素を生成)  
	∟ get_diff_file1(差分を出す1つ目のファイルの選択時処理)  
	∟ get_diff_file2(差分を出す2つ目のファイルの選択時処理)  
	∟ select_file(ファイル選択時の処理)  
	∟ preview(treeviewへの表示処理)  
	∟ diff_preview(差分を表示する処理)  
	∟ check_diff(差分を判定する処理)  
	∟ diff_only_preview(差分のみを表示するフラグ管理)  
	∟ change_content(セグメントボタンでの切り替えを行い、表示しているファイルを切り替える)  
	∟ change_mode(サイドメニュー下部にあるモードチェンジ処理)  

</details>

## 簡易テスト
### ■正常系
1. 1つ目のファイル選択→セグメントボタンの表示が1つ目のファイル克、ツリービューにテキスト内容が表示される  
2. 2つ目のファイル選択→セグメントボタンの表示が2つ目のファイル克、ツリービューにテキスト内容が表示される  
3. 1.2.を実行後、差分部分が薄い赤色で表示される  
4. 1.2を実行後セグメントボタンによるツリービューの表示の切り替えが正常に実行される
5. 1.2.を実行後、差分のみ表示ボタンを押下し、ツリービューの表示が差分のみとなる
6. 6.を実行後、差分のみ表示ボタンがすべて表示ボタンに切り替わる → すべて表示ボタン押下で差分以外も表示される

### ■異常系
1. ファイルを選択していない状態で差分のみ表示ボタン押下 → エラー出力
